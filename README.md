## 🛠️ Troubleshooting

If the software does not work, try these steps:

* Run as Administrator: Right-click the shortcut and select Run as administrator. This grants the tool permission to read your Wand files.
* Check Updates: Visit the releases page often to download newer versions. Old versions might lose compatibility with the Wand app if that software updates.
* Restart: Close the Wand app and the enhancement tool, then restart both applications.
* Permissions: Ensure your antivirus software does not block the Wand-Enhancer. You may need to add an exception for the folder where you installed the tool.
